# React Project - StageLink
SpringBoot, 리액트를 활용한 공연 예매 및 커뮤니티 사이트의 프론트입니다.

# 개발 기간
2025.04.28 ~ 2025.06.13

# 맴버구성
1. 김용현 - 공연 관리, 예매 관리, 통계, 로그인 보안
2. 이택동 - 회원 관리, 신고
3. 전에스더 - 공지사항 및 Q&A, 로그인 토큰

# 프로젝트 경험
## 가장 힘들었던 점
- 기획 단계에게 까먹은 점에 대한 후속 조치가 힘들었다. KOPIS 사용시, 데이터의 종류와 형식를 보았으나, 출력 방식을 고려하지 못하였다. (KOPIS 데이터 출력 : 공연 정보 검색시, 공연장 출력/ 프로젝트 데이터 저장 방식 : 공연장 검색시, 공연 정보 출력)

## 가장 좋았던 점
- 요구 사항, WBS 등 문서를 작성하여 최대한 팀원들과 논쟁을 기획단계에서 줄었다.

## 배운 점
- 요구 사항을 더 꼼꼼하게 챙기게 되었다.
- 명확히 적힌 문서면 팀 프로젝트도 간단히 될 줄 알았으나, 서로 이해하는 방향이 다를 수 있음을 깨달았다. 결국 팀원간의 교류가 중요하다.
